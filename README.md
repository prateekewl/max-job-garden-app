# Max's Job Garden — public shell

This repository contains the privacy-safe, Max-only web app for Max's Job Garden.

- No CV, contact details, application history, Google Sheet data, backend code, API credentials, or access tokens are stored here.
- The workspace opens directly and saves Max's jobs, preferences, and progress on his device.
- A curated public jobs file contains only live advert data from selected official employer boards.
- Hard filters admit only Max's CV-backed role families in Glasgow, Edinburgh, or genuinely UK-accessible remote locations.

The private source repository verifies and allow-lists every file before publishing here.
