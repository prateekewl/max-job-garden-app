# Max's Job Garden — public shell

This repository contains only the privacy-safe static shell for Max's Job Garden.

- No CV, contact details, application history, Google Sheet data, backend code, API credentials, or access tokens are stored here.
- The workspace unlocks only when the separate authenticated backend issues a personal link.
- The sample-data preview is available at `?demo=1`.

The private source repository verifies and allow-lists every file before publishing here.
